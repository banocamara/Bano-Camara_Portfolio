# Portfolio Personnel

Bienvenue sur mon portfolio personnel développé en HTML, CSS et PHP.

👉 Voir le site en ligne :  
https://bano-camara.wuaze.com/

## Description

Ce projet présente mes compétences, mes projets réalisés ainsi que mon évolution dans le domaine du développement web et de la cybersécurité.

Dans cette version du portfolio, plusieurs fonctionnalités ont été ajoutées :

- intégration du PHP ;
- formulaires dynamiques avec validation ;
- système de recherche de projets ;
- organisation du site avec plusieurs pages ;
- amélioration du design responsive ;
- ajout d’images et de vidéos.

## Hébergement

Le portfolio n’est plus hébergé sur GitHub Pages, car GitHub Pages ne prend pas en charge PHP.

Le projet est désormais hébergé sur InfinityFree afin de permettre l’utilisation des fonctionnalités dynamiques du site.

## Base de données

Une base de données sera intégrée prochainement afin d’ajouter de nouvelles fonctionnalités dynamiques au portfolio.
